# Antora Transition

This repository is in a transitional state:

- legacy Markdown authoring remains under `docs/`
- Antora is now the target site engine
- generated Antora source lives under `.antora-src/`
- CI converts public articles from Markdown to AsciiDoc during build
- public article URLs are expected to be slug-driven, not filename-driven

Legacy MkDocs files remain temporarily until cleanup completes.
